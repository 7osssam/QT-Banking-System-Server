\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classUserInitRequest}{User\+Init\+Request}} class handles user initialization requests for User Widget and Admin Widget. 

This class processes requests to initialize user information based on their credentials. It retrieves user details and account information if applicable, and prepares a response with the user\textquotesingle{}s role and associated data.
