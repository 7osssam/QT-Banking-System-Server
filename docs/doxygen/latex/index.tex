 \doxysection*{Overview}

QT-\/\+Banking-\/\+System-\/\+Server is the server-\/side implementation of a banking system, developed using the Qt 6/\+C++ framework. The server handles client requests, performs database operations using Postgre\+SQL with Supabase hosting, and ensures secure and efficient communication between clients and the database.

The server application is designed to be multi-\/threaded, enabling concurrent handling of multiple clients and requests. It includes features such as request handling, database operations, logging, and secure communication to provide a robust and reliable banking system.

\begin{quote}
\mbox{[}!\+NOTE\mbox{]} This project is part of a Banking System (TCP Client-\/\+Server) project, which includes the following repositories\+:
\begin{DoxyItemize}
\item \href{https://github.com/7osssam/QT-Banking-System-Client}{\texttt{ QT-\/\+Banking-\/\+System-\/\+Client}}\+: The client-\/side implementation of the banking system.
\item \href{https://github.com/7osssam/QT-Banking-System-Server}{\texttt{ QT-\/\+Banking-\/\+System-\/\+Server}}\+: The server-\/side implementation of the banking system. 
\end{DoxyItemize}\end{quote}


\doxysection*{Features}


\begin{DoxyItemize}
\item {\bfseries{\mbox{\hyperlink{classRequest}{Request}} Handling\+:}} Manages various client requests such as user management, transaction processing, and data retrieval.
\item {\bfseries{Database Operations\+:}} Performs CRUD operations on user accounts and transaction records.
\begin{DoxyItemize}
\item By using the \href{src/dbModule}{\texttt{ Database Module}}, the server can connect to the database, execute queries, and handle results in a multi-\/threaded environment using Qt\textquotesingle{}s {\ttfamily QThread\+Storage} for thread-\/local storage.
\end{DoxyItemize}
\item {\bfseries{Logging\+:}} Logs server activities and client requests for monitoring and debugging purposes in a multi-\/threaded fashion.
\begin{DoxyItemize}
\item For more details, see the \href{https://github.com/7osssam/LogByThread}{\texttt{ Log\+By\+Thread}} library used in the project.
\end{DoxyItemize}
\item {\bfseries{Secure Communication\+:}} Implements security measures to ensure secure communication with clients and database.
\begin{DoxyItemize}
\item For example, hashing and encryption techniques are used to protect sensitive data.
\end{DoxyItemize}
\item {\bfseries{Concurrency server\+:}} The server can handle multiple clients concurrently using Qt\textquotesingle{}s {\ttfamily QTcp\+Server} and {\ttfamily QTcp\+Socket} classes.
\begin{DoxyItemize}
\item The server can process multiple requests simultaneously using a thread pool.
\end{DoxyItemize}
\end{DoxyItemize}
