\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classServerHandler}{Server\+Handler}} class handles incoming client connections in a separate thread. 

This class is responsible for managing communication with a client, verifying data integrity using cryptographic hashes, and processing requests through the \mbox{\hyperlink{classRequestManager}{Request\+Manager}}. It operates in a separate thread to ensure that the server remains responsive.
