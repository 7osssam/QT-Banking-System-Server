\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classRequestManager}{Request\+Manager}} class handles incoming requests and dispatches them to the appropriate request handler. 

The \mbox{\hyperlink{classRequestManager}{Request\+Manager}} class is responsible for managing and processing different types of requests. It maintains a map of request handlers for various request types, processes incoming request data, and generates appropriate responses. The class also includes error handling for JSON parsing issues.
